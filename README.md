# Arduino LED Chaser (Tinkercad Simulation)

## Overview
This project simulates an LED chaser using Arduino in Tinkercad.
It demonstrates basic output sequencing and timing control.

## Components
- Arduino Uno
- LEDs
- Resistors

## How It Works
Each LED turns ON one at a time in sequence using digital outputs.
A delay controls the speed of the running light effect.

- Repetitive code structure
